# Pre-Requirements:

- Python 3 
- Pip
- Django


# Installation

- Cd to repo main folder 
- Execute commands in order: 
    >python -m venv venv
    >venv\Scripts\activate
    >pip install django

# Run Server:

- Cd to repo main folder and then sucursal_crud
- Execute command: >python manage.py runserver
- Then go to browser at "http://localhost:8080"
